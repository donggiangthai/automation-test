## Final Assessment Structure

### Part 1: Written Test (30 min)

**Topics covered:**
- Python fundamentals
- pytest concepts
- Playwright basics
- AI-assisted testing practices

**Format:** Multiple choice + short answer

---

### Part 2: Practical Assessment (60 min)

**Task:** Create an E2E test suite for a given feature

**Requirements:**
- Minimum 5 test cases
- Use Page Object Model
- Use network mocking
- Use parametrization
- Include fixtures
- Generate test report
