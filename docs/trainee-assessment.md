# Automation Test Trainee Assessment

**Purpose:** Evaluate trainee's starting level to customize the training plan  
**Duration:** 30-45 minutes  
**Format:** Interview + Quick practical tasks

---

## Instructions for Interviewer

1. Create a comfortable atmosphere - this is not an exam
2. Note responses and observations
3. Score each answer: 0 (no answer), 1 (basic), 2 (good)
4. Calculate total score at the end to determine training level

---
